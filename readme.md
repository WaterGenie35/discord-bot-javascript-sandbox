## 2. Usage

### 2.1. Run

```bash
npm run start
```

### 2.2. Development

```bash
# Watch
npm run dev

# Test
npm run test

# Build
npm run build

# Scripts
# e.g. deploy-commands
npm run script script-name
```

#### 2.2.1. Scripts

```bash
# Register new commands and update existing ones
npm run script deploy-commands
```
